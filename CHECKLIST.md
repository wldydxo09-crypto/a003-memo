# ✅ 스마트 업무 비서 - 최종 점검 체크리스트

앱 배포 전, 아래 기능이 정상 작동하는지 확인해주세요.

## 1. 🔑 필수 설정 확인 (가장 중요)
- [ ] **.env.local 파일**: `GEMINI_API_KEY`가 올바르게 입력되어 있나요? (비어있거나 `your_key_here`라면 안 됩니다!)
- [ ] **Firebase 규칙**: Firestore Database Rules가 `allow read, write: if true;`로 설정되어 있나요? (콘솔에서 확인)

## 2. 🧪 기능 테스트
- [ ] **로그인**: Google 로그인이 정상적으로 되나요?
- [ ] **메뉴 이동**: 왼쪽 사이드바 메뉴를 클릭했을 때 제목이 바뀌나요?
- [ ] **AI 요약**: 내용을 입력하고 `AI 요약` 버튼을 눌렀을 때 요약이 생성되나요?
- [ ] **저장**: 내용 입력 후 `저장하기`를 눌렀을 때, 아래 리스트에 카드가 추가되나요?
- [ ] **UI 확인**: 스크롤 할 때 헤더가 콘텐츠를 가리나요? (배경색이 있어서 글자가 안 겹쳐야 정상)

## 3. 🚀 배포 준비
위 테스트가 모두 통과했다면 배포할 준비가 된 것입니다.
터미널에서 다음 순서로 배포하세요:

1. `firebase login`
2. `firebase init hosting` (Existing project 선택 -> `a001-todo...` 선택 -> `out` 폴더가 아닌 `.next`설정 주의 필요)
   * **Tip**: Next.js 배포는 조금 까다로울 수 있으니, 테스트 완료 후 에이전트에게 "배포해줘"라고 말씀해주세요!
